# y-farnese-tattoo-webpage

Marketing and portfolio website for **Yolanda Farnese**, Tattoo Artist (Fine Line & Blackwork) in Uberlandia - MG.

## Tech Stack

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS
- Prismic CMS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Root `/` redirects to `pt-br` (default locale hidden from URL).

## Header Controls

- **Locale dropdown:** HeroUI-style dropdown showing current language (Português/English). No label.
- **Theme switch:** Sun icon in light mode, moon icon in dark mode. Click to toggle. No label.
